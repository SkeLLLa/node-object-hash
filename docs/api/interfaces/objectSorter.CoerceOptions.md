### bigint

• `Optional` **bigint**: `boolean`

If `true` converts BigInt to string

**`Example`**

```ts
// coerce.bigint = true
1n === '1';
```

**`Example`**

```ts
// coerce.bigint = false
1n !== '1';
```

**`Default`**

```ts
true;
```
